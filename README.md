# Login Authentication Web App

A simple front-end **Login Authentication System** built using **HTML**, **CSS**, and **JavaScript**. It includes a registration page, login page, and a protected dashboard that simulates user authentication on the client side.

---

## Objective

To create a basic authentication flow that allows users to register, log in, and access a secure dashboard only after successful login. This project demonstrates form handling, input validation, and basic client-side session control.

---

## Features

- User Registration form  
- Login form with basic validation  
- Protected dashboard page (accessible only after login)  
- Simple session simulation using localStorage  
- Responsive layout for all devices  

---

## Tools Used

- HTML5 – for page structure  
- CSS3 – for styling and layout  
- JavaScript – for validation, authentication logic, and navigation  
- Visual Studio Code – as the code editor  
- Web Browser – for testing and debugging  

---

## Steps Performed

1. Created individual HTML pages for Login, Register, and Dashboard  
2. Styled each page using CSS for a clean, modern look  
3. Used JavaScript to:
   - Validate registration inputs  
   - Store user credentials in localStorage  
   - Authenticate users on login  
   - Restrict dashboard access unless logged in  
4. Added redirect logic to simulate security  
5. Tested on various devices and fixed layout/logic issues
